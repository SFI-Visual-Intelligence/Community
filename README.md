# Visual Intelligence Community

> **Visual Intelligence** is a Centre for Research-based Innovation (SFI), funded by the [Research Council of Norway](https://www.forskningsradet.no/) (project #309439) and consortium partners.

**Website:** https://www.visual-intelligence.no

---

## About

Visual Intelligence aims to be the lead provider of novel deep learning-based solutions for complex image analysis — addressing shared innovation needs across corporate and public partners, and beyond.

We bring together researchers, industry, and public institutions to push the boundaries of what is possible with visual data.

**Operating period:** 2020–2026

---

## Mission & Vision

We develop cutting-edge deep learning methods that make image analysis more capable, trustworthy, and explainable — across sectors that rely on visual data to make high-stakes decisions.

| Area | Description |

| **Energy & Industry** | Remote inspection, infrastructure monitoring, industrial vision |
| **Earth Observation** | Satellite imagery analysis, environmental monitoring |

---

## Key Research Challenges

- **Learning from limited data** — robust models when labelled data is scarce
- **Context and dependencies** — exploiting spatial, temporal, and semantic structure
- **Uncertainty quantification** — knowing when a model does not know
- **Explainability & reliability** — interpretable and trustworthy deep learning
